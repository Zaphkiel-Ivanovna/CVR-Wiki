# CVR Animator Driver

Ce composant peut être utilisé pour contrôler les paramètres d'un [Animator](https://docs.unity3d.com/ScriptReference/Animator.html) d'une autre animation. Utile pour, par exemple, créer des sous-animators et contrôler leurs paramètres avec l'animateur principal.

<img src={useBaseUrl('/img/composants-example/cvr-animator-driver.png')} alt="1" />

### Animator Parameters

Une liste des paramètres de l'animateur que vous voulez contrôler.

##### Animator

Sélectionnez l'[Animator](https://docs.unity3d.com/ScriptReference/Animator.html) que vous voulez contrôler pour ce paramètre spécifique.

##### Parameter

Tous les paramètres de l'[Animator](https://docs.unity3d.com/ScriptReference/Animator.html) selectionné seront répertoriés ici.

##### Value

Définissez la valeur du paramètre ici. Cette propriété peut être modifiée pendant l'exécution, par exemple par une animation.

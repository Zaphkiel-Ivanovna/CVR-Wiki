# CVR Audio Driver

Ce composant vous permet de basculer entre plusieurs clips audio pour une source audio.  
Ceci est recommandé et utile lorsque vous utilisez de nombreux clips audio pour le même emplacement de source audio et les mêmes paramètres.

### Propriétés
<hr />

##### Audio Source

Sélectionnez l'[Audio Source](https://docs.unity3d.com/ScriptReference/AudioSource.html) sur laquelle les clips doivent être lus.

##### Audio Clips

Une liste d'[Audio Clip](https://docs.unity3d.com/ScriptReference/AudioClip.html) entre lesquelles vous voulez passer.

##### Selected Audio Clip

L'index du clip audio qui est actuellement sélectionné. Il peut être modifié pendant l'exécution, par exemple par une animation.

##### Play On Switch

Si cette option est activée, le clip audio sera lu lors de la commutation du clip audio sélectionné.

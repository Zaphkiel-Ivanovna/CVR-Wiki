# CVR Asset Info

Ce composant permet d'associer un Avatar/World/Props à un objet en ligne sur les serveurs ChilloutVR à l'aide d'un identifiant unique propre à l'objet.

<img src={useBaseUrl('/img/composants-example/cvr-asset-info.png')} alt="1" />

### Propriétés
<hr />

##### Identifiant unique

Identifiant lié à l'objet sur les serveurs ChilloutVR

:::caution Avertissement
Ce composant étant dépendant du [CVR Avatar](../cck_composants/cvr-avatar) ou [CVR World](../cck_composants/cvr-world), celui-ci ne doit pas être utilisé seul.
:::
